# Math Master
In this repository, you find the project Math Master. This project started as a school project, that I worked further on in my free time. The idea for Math Master came from me to build a school subject into a new learning experience. I wanted to build a online platform where anyone can learn math and further their math skills.

## Features
What makes Math Master different from other learning platforms is that users are playing interactive games to learn and further their math skills. 

## How To Use
This web application is useable on the website of Math Master. 

## Technologies
Math Master is build as a Single Page Webapp in JavaScript. The Back-End is build with Express. The application makes use of a database build in postgres.
